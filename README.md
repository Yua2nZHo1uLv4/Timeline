# 仓库说明
神岛天梯S3《蝴蝶》仓库
# 策划案
## 玩法介绍
游戏采用LIM-Yua2nZHo1uLv4(13048840，即项目拥有者)的神岛2D引擎   
地图预设
每个章节内存在多个地点相同，时间不同的分支   
部分分支内存在通往其他时间点的传送门   
玩家可以多人合作完成解密   
## 机制介绍
每个分支之间设定的时间相差1年(或多年)   
位于过去的玩家会默认一直呆在现有位置，未来的玩家可以看到过去玩家的白骨   
过去的玩家可以实时影响未来的时间线   
过去玩家的物品会随着时间放到玩家的白骨旁边   
玩家可以影响物品，也可以通过物品影响地图(或直接影响)   
所有的物品和地图部件都有互动时间限制，超出时间限制则显示已经损坏或还未诞生   
物品可以被带到其他时间   
## 剧情
略，以游戏具体为准
